## 🔧 需要您完成的設置

### 1. 在 Google Ads 中建立新的轉換動作

#### 步驟 1: 登入 Google Ads
1. 前往 [ads.google.com](https://ads.google.com)
2. 登入您的帳戶 (帳戶ID: AW-879825412)

#### 步驟 2: 建立購買轉換動作
1. 點擊 **工具和設定** → **轉換**
2. 點擊 **+ 新增轉換動作**
3. 選擇 **網站**
4. 設定如下：
   ```
   轉換名稱: 購買完成
   類別: 購買
   價值: 使用不同的價值 (動態價值)
   計算方式: 每次轉換
   轉換期間: 30天
   瀏覽後轉換期間: 1天
   歸因模型: 資料驅動
   ```
5. 點擊 **建立並繼續**
6. **重要**: 複製轉換標籤 (格式: `AW-879825412/XXXXXXXXX`)

#### 步驟 3: 建立開始結帳轉換動作
1. 重複上述步驟，但設定為：
   ```
   轉換名稱: 開始結帳
   類別: 其他
   價值: 使用不同的價值 (動態價值)
   計算方式: 每次轉換
   ```

### 2. 更新網站代碼中的轉換標籤

#### 需要替換的標籤位置

**在 `cart.html` 中**:
```javascript
// 第 1724 行附近，找到這行：
'send_to': 'AW-879825412/BEGIN_CHECKOUT_LABEL',

// 替換為您從 Google Ads 複製的實際標籤：
'send_to': 'AW-879825412/您的開始結帳轉換標籤',
```

**在 `order_confirmation.html` 中**:
```javascript
// 第 875 行附近，找到這行：
'send_to': 'AW-879825412/PURCHASE_LABEL',

// 替換為您從 Google Ads 複製的實際標籤：
'send_to': 'AW-879825412/您的購買轉換標籤',
```

### 3. 測試轉換追蹤

#### 使用 Google Tag Assistant
1. 安裝 [Google Tag Assistant Legacy](https://chrome.google.com/webstore/detail/tag-assistant-legacy-by/kejbdjndbnbjgmefkgdddjlbokphdefk) Chrome 擴充功能
2. 開啟您的網站
3. 點擊擴充功能圖示
4. 檢查是否顯示 Google Ads 代碼

#### 實際測試流程
1. **測試 Line 聯繫轉換**:
   - 在任何頁面點擊聯繫按鈕
   - 檢查瀏覽器控制台是否顯示 "🎯 追蹤Line聯繫轉換"

2. **測試開始結帳轉換**:
   - 添加商品到購物車
   - 進入購物車頁面
   - 填寫客戶資訊並點擊 "建立訂單"
   - 檢查控制台是否顯示 "🎯 追蹤開始結帳轉換"

3. **測試購買轉換**:
   - 完成上述步驟
   - 確認訂單後會跳轉到訂單確認頁面
   - 檢查控制台是否顯示 "🎯 追蹤購買轉換"

### 4. 驗證轉換數據

#### 在 Google Ads 中檢查
1. 前往 **工具和設定** → **轉換**
2. 查看各轉換動作的狀態
3. 等待 24-48 小時後檢查轉換數據

#### 預期結果
- Line 聯繫轉換應該立即顯示
- 開始結帳和購買轉換需要完整測試流程

   - 整合 Facebook Pixel

## ⚠️ 重要注意事項
