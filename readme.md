# DeepVape - 頂級電子菸專賣店

## 網站結構

### 首頁佈局（已重新設計）
1. **打字樣式的公告區塊** - 動態顯示重要公告
2. **標題區塊** - 包含主標題、副標題、描述文字及按鈕
   - 立即探索按鈕
   - 重要通知按鈕（彈跳視窗）
3. **輪播圖區塊** - 固定高度，純圖片展示（無文字）

### 主要功能
- 響應式設計，支援桌面、平板、手機
- 年齡驗證系統
- 購物車功能
- 產品展示與管理
- CMS 後台管理

### CMS 後台管理
- 公告管理
- 輪播圖管理（純圖片，無文字輸入）
- 重要通知管理
- 聯繫信息管理
- 價格管理

### 技術特色
- Netlify CMS 整合
- 無伺服器函數
- 現代化 CSS 動畫效果
- 賽博朋克風格設計

## 部署
網站部署在 Netlify 平台，支援自動部署和 CMS 管理。
